# React Practice Playground

This is a beginner-friendly React + Vite project for daily practice.

## What is inside

- A counter for `useState` practice
- A task tracker for forms, lists, and conditional rendering
- A color picker for controlled inputs and live styling
- A simple layout you can keep extending with your own components

## Run the project

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Build the project

```bash
npm run build
```

## Practice ideas

- Split each section into its own component
- Add delete buttons for tasks
- Save tasks to `localStorage`
- Add React Router and a second practice page
